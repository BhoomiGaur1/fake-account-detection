# Fake Instagram Account Detection (Machine Learning)

## Overview

Fake accounts on social media platforms are a major problem as they spread spam, misinformation, and malicious content.
This project uses **Machine Learning techniques** to automatically detect whether an Instagram account is **real or fake** based on different profile features.

The model analyzes various characteristics of an account and predicts the probability of the account being fake.

---

## Objective

The objective of this project is to build a **classification model** that can identify fake Instagram accounts using profile-based features.

---

## Features Used

The model uses several Instagram profile attributes such as:

* Number of followers
* Number of following
* Number of posts
* Username length
* Bio length
* Profile picture presence
* External URL presence

These features help the model distinguish between genuine and fake accounts.
